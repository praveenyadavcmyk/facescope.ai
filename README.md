# FakeScope AI 📰

AI-powered Fake News Detection Web Application built using Flask, Machine Learning, JavaScript, HTML, and CSS.

## Features
- Real-time fake news prediction
- Clean modern UI
- Machine learning model integration
- Flask backend API
- Character counter
- Responsive frontend

## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python3 App.py
